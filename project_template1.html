프레스캣

PRESSCAT

반응형 웹사이트로 제작하여 PC, 모바일, 태블릿 등 다양한 디바이스 화면에서 최적의 콘텐츠를 이용할 수 있도록 제작하였습니다.
웹 에이전시인 프레스캣은 사용자에게 신뢰감을 높이기 위해, 기존의 복잡하고 정돈되지 않은 레이아웃을 간결하게 구성하여 콘텐츠 경험을 개선하고 깔끔한 느낌을 높였습니다.

PC Web

메뉴별 콘텐츠를 분석하여 각기 다른 레이아웃으로 페이지 디자인을 기획하였으며 메뉴마다의 차별성을 주려고 하였습니다.
사용자들이 기기에 구애받지 않고 어떠한 환경에서도 동일한 경험을 받을 수 있도록 제작하였으며
사용자의 접근성을 강화하고 편의를 제공하였습니다.

Mobile App

이용자가 방문한 목적을 효과적으로 수행하기 위해 모바일 화면에서의 사용성을 고려하고
그 특징을 활용하여 컨텐츠 내용을 쉽고 빠르게 접할 수 있도록 UI를 구성하고 디자인하였습니다.
컨텐츠 중심으로 구성된 레이아웃과 적절한 인터렉션 효과는 이용자의 흥미를 더해줍니다.

Mobile Page

사용자가 정보를 빠르게 인지 할 수도록
상단에 지속적으로 노출되는 메뉴를 배치하고
콘텐츠 분리와 중요도에 따른 배치로 사용자 중심으로 디자인하였습니다.
